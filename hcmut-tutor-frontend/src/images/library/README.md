# Library Banner Images

Vui lòng lưu 3 hình ảnh banner vào thư mục này với các tên sau:

1. `banner1.jpg` - Hình trường Đại học Bách Khoa (bên ngoài)
2. `banner2.jpg` - Hình thư viện (bậc thang ngồi đọc sách)
3. `banner3.jpg` - Hình phòng đọc sách (kệ sách và bàn ghế)

Các hình này sẽ được sử dụng cho slideshow banner trên trang chủ thư viện.
Banner sẽ tự động xoay vòng mỗi 5 giây.
